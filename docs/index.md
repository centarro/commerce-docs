# Drupal Commerce docs

Drupal Commerce is an open-source eCommerce platform that lets you sell any type of product to anyone in the world. It powers _billions_ in sales across thousands of online stores.

Building on the platform requires general knowledge about how to install and manage Drupal on a web server. Once you have Drupal installed, adding the Commerce Core project to it gives you a foundation of eCommerce data objects, user interfaces, and subsystems.
